Returns

Maximum or minimum value of phase.

Return type

float

property phase

Sets the phase of the internal modulation source when modulating by the internal source with shape Sinusoid, Square, Ramp, NRamp or Triangle.

Access

ReadWrite

Type

float

Remarks
•Parameter : From -360 to 360 (Degree)
From -2π to 2π (Radian)
From -arbitrary waveform period to +arbitrary waveform period (Second)
•Typical Return : +3.000000000000000E+01
•This command applies only to the 33600 Series and arbitrary waveforms of at least 64 samples.
•The default units of the phase are set by PhaseUnit.
