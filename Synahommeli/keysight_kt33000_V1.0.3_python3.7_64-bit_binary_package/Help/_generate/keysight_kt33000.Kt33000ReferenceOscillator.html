class Kt33000ReferenceOscillator

Kt33000ReferenceOscillator interface.

property auto_enabled

Disables(OFF) or enables(ON) automatic selection of the reference oscillator. Default ON.

Access

ReadWrite

Type

State

Remarks
•This command is supported on 33500 and 33600 series only.
•ON: the instrument preferentially selects a 10 MHz signal from the rear-panel 10 MHz In connector.
•OFF: the instrument selects the reference oscillator based on ReferenceOscillatorSource setting.
property current_source

Indicates which reference oscillator signal is currently in use.

Access

ReadOnly

Type

ReferenceOscillatorSource

Remarks
•This command is supported on 33500 and 33600 series only.
•Typical Return : Internal) or External (10 MHz connector on rear panel).
•The basic difference between this command and ReferenceOscillatorSource query command is that, this command returns the Reference signal source which is selected after enabling automatic selection of the reference oscillator(i.e ReferenceOscillatorAutoEnabled ON) while ReferenceOscillatorSource query command returns the Reference signal source which is selected by you. For example if you select External Reference Oscillator source using ReferenceOscillatorSource command then both ReferenceOscillatorSource query command and Current Source query command will return "EXT" but if you enable automatic selection of Reference Oscillator signal source, then ReferenceOscillatorSource query command will return "EXT" while Current source query command returns "INT".
•INT refers to the basic internal TCXO or the optional ovenized OCXO oscillator, whichever is installed.
property source

Selects the source for the reference oscillator used as the frequency/phase reference for signals generated by the instrument. Selects either Internal or External source. Default Internal.

Access

ReadWrite

Type

ReferenceOscillatorSource

Remarks
•This command is supported on 33500 and 33600 series only.
•Internal: instrument uses the internal reference oscillator and ignores the signal at the 10 MHz In connector.
•
External: instrument uses signal on the rear-panel 10 MHz In connector as reference, and generates an error if this signal is absent or the instrument cannot lock to it. In such error cases, instrument output continues, but the frequency will be unstable.
