TriggerSource

class TriggerSource

Selects the Trigger Source.

Members:

CHANNEL1 : Channel1. This is default OutputTriggerSource.

CHANNEL2 : Channel2

BUS : When the Bus (software) source is selected, the function generator outputs one burst each time a bus trigger command is received.

IMMEDIATE : When the Immediate (internal) source is selected, the function generator outputs continuously when the burst mode is enabled. This is default TriggerSource.

EXTERNAL : When the External source is selected, the function generator will accept a hardware trigger applied to the rear-panel Trig In connector.

TIMER : When the Timer source is selected, the trigger events are spaced by a timer with the first trigger as soon as the INIT signal occurs. Only applies to BurstMode Triggered.
