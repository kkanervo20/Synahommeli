query_output_configuration(self: keysight_kt33000.keysight_kt33000.Kt33000OutputChannelApply) → Tuple[keysight_kt33000.keysight_kt33000.FunctionShape, float, float, float]

Query the function generator’s current configuration and return a quoted string.

Returns

A tuple containing:

  • function (FunctionShape) : Output Function

  • frequency (float) : Frequency of the current output function.

  • amplitude (float) : Amplitude of the current output function.

  • offset (float) : Offset of the current output function.

Type

tuple[FunctionShape, float, float, float]

Remarks
The function, frequency, amplitude, and offset are returned as shown above. The amplitude, but not the offset, is returned as specified by VoltageUnit.
set_arb_waveform(self: keysight_kt33000.keysight_kt33000.Kt33000OutputChannelApply, sample_rate: float, amplitude: float, offset: float) → None

Outputs arbitrary waveform, using the specified sample rate, amplitude, and offset.

Parameters

  • sample_rate (float) – Sample Rate from 1 µSa/s to 250 MSa/s (EDU33210 series and 33500 Series) or 1 GSa/s (33600 Series), default 40 kSa/s.

  • amplitude (float) – Amplitude from 1 mVpp to 10 Vpp into 50 Ω, 2 mVpp to 20 Vpp into an open circuit, default 100 mVpp into 50 Ω.

  • offset (float) – DC offset voltage (default 0), from ±5 VDC into 50 Ω, or from ±10 VDC into an open circuit.

Returns

None

Remarks
General
•Setting a sample rate when not in the ARB mode will not change the frequency. For example, if the current function is sine, setting sample rate has no effect until the function changes to ARB.
•High sample rates may affect the actual amplitude due to filter roll-off.

Options
•See SelectArbWaveform for available arbitrary waveform options. With SelectArbWaveform, you may select a built-in arbitrary waveform or the waveform currently downloaded to volatile memory using Memory commands.

Offset Voltage
•The relationship between offset voltage and output amplitude is shown below. Vmax is the maximum peak voltage for the selected output termination (5 V for a 50 Ω load or 10 V for a high-impedance load).
|Voffset| < Vmax - Vpp/2
If the specified offset voltage is not valid, the instrument will adjust it to the maximum DC voltage allowed with the specified amplitude. From the remote interface, a "Data out of range" error will also be generated.
•Limits Due to Output Termination: The offset range depends on the output termination setting. For example, if you set offset to 100 mVDC and then change output termination from 50 Ω to "high impedance," the offset voltage displayed on the front panel doubles to 200 mVDC (no error is generated). If you change from "high impedance" to 50 Ω, the displayed offset voltage will be halved. See OutputLoad for details.

set_prbs(self: keysight_kt33000.keysight_kt33000.Kt33000OutputChannelApply, bit_rate: float, amplitude: float, offset: float) → None

Outputs a pseudo-random binary sequence with the specified bit rate, amplitude and DC offset.

Parameters

  • bit_rate (float) – Bit rate in bits/s, default 1000.

  • amplitude (float) – Desired output amplitude in Vpp, Vrms or dBm. It can be from 1 mVpp to 10 Vpp into 50 Ω, or twice that into an open circuit. Default is 100 mVpp into 50 Ω.

  • offset (float) – DC offset voltage (default 0), from ±5 VDC into 50 Ω, or from ±10 VDC into an open circuit.

Returns

None

Remarks
Frequency
•PRBS is generated by a Maximum Length Sequence (MLS) generator (Linear Feedback Shift Register) which may be configured to several standard configurations. Default is PN7 at 1000 bits/second.
•A PRBS waveform using polynomial PNx is generated by a shift register of x bits, and the output waveform begins with x sample periods of high output. Sample period is the reciprocal of the sample rate (PRBSBitRate), and the channel's Sync pulse indicates the waveform's start. For example, if the PRBS uses PN23 with sample rate 500 Hz, the output begins with 46 ms of high output (23 x 2 ms).
•Unlike the SetNoise command, this command operates with the Sync output enabled.
•The Sync function indicates the beginning of the Pseudo-random function sequence.

set_pulse(self: keysight_kt33000.keysight_kt33000.Kt33000OutputChannelApply, frequency: float, amplitude: float, offset: float) → None

Outputs a pulse wave with the specified frequency, amplitude, and DC offset.

Parameters

  • frequency (float) – Frequency in Hz, default 1 kHz.

  • amplitude (float) – Desired output amplitude in Vpp, Vrms or dBm. It can be from 1 mVpp to 10 Vpp into 50 Ω, or twice that into an open circuit. Default is 100 mVpp into 50 Ω.

  • offset (float) – DC offset voltage (default 0), from ±5 VDC into 50 Ω, or from ±10 VDC into an open circuit.

Returns

None

Remarks
•It performs the following operations:
 •Preserves either the current pulse width setting (PulseWidth) or the current pulse duty cycle setting (PulseDutyCycle).
 •Preserves the current transition time setting (PulseTransitionEdgeTime).
 •May cause instrument to override the pulse width or edge time setting to comply with the specified frequency or period (PulsePeriod).

Frequency
•The APPLy command must be appropriate for the function. For example, setting frequency 300 MHz results in a "Data out of range" error. In that case, the frequency would be set to the instrument's maximum frequency for a pulse.
