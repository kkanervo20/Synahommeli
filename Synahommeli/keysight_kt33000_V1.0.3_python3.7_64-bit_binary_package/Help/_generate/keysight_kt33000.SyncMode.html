SyncMode

class SyncMode

Specifies normal Sync behaviour (Normal), forces Sync to follow the carrier waveform (Carrier), or indicates marker position (Marker).

Members:

NORMAL : Normal causes SYNC output to follow in various modes when frequency control is fixed, in Sweep, Burst or Modulation modes. This is default Sync Mode.

CARRIER : Carrier causes SYNC output always to follow the carrier wave even when in Sweep, Burst, and modulation modes.

MARKER : In marker mode, sync always follows current when in CW mode and function is arbitrary, or Burst state is ON or Sweep is ON.
