class Kt33000OutputChannelFSK

Kt33000OutputChannelFSK interface.

property enabled

FSK modulation state (Enabled or Disabled) for the specified channel.

Access

ReadWrite

Type

bool

Remarks
•Parameter : 0 (OFF) or 1 (ON). Default 0(OFF).
•To avoid multiple waveform changes, enable modulation after configuring the other modulation parameters.
•Only one modulation mode may be enabled at a time.
•The instrument will not enable modulation with sweep or burst enabled. When you enable modulation, the sweep or burst mode is turned off.

property internal_rate

Sets the rate at which output frequency “shifts” between the carrier and hop frequency.

Access

ReadWrite

Type

float

Remarks
•Parameter : 0.125 mHz to 1 MHz, default 10 Hz.
•Typical Return : +1.000000000000000E+01.
•The FSK rate is used only with the internal source (i.e FSKSource Internal).
•The modulating waveform is a square wave with a 50% duty cycle.

property source

Selects the source of the modulating signal. FSKey cannot accept CH1 or CH2.

Access

ReadWrite

Type

ModulationSource

Remarks
•Parameter: Internal or External. Default Internal.
•Typical Return : Internal or External.
•With External source, the output frequency (FSK) is determined by the signal level on the rear-panel Ext Trig connector. When a logic low is present, the carrier frequency is output. When a logic high is present, the hop frequency is output.
•With Internal source, the rate at which output frequency (FSKey)"shifts" between the carrier frequency and the alternate frequency is determined by the FSK rate.
•The connector used for externally-controlled FSK waveforms (Trig In) is not the same connector that is used for externally-modulated AM, FM, PM, and PWM waveforms (Modulation In). When used for FSK, the Trig In connector does not have adjustable edge polarity and is not affected by the TriggerSlope command.
•A channel may not be its own modulation source.
