DisplayARBRateUnits

class DisplayARBRateUnits

Selects the arbitrary waveform rate units.

Members:

S_RATE : Sets the arbitrary waveform rate units as samples per second. This is default Arbitrary Rate unit.

FREQUENCY : Sets the arbitrary waveform rate units as Hz.

PERIOD : Sets the arbitrary waveform rate units in seconds.
