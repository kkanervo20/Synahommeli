StatusByteFlags

class StatusByteFlags

StatusByteFlags enumeration with values corresponding to the 8-bit status byte.

Members:

STANDARD_EVENT_SUMMARY : Bit 5, value 32, of the Status Byte Register indicating Standard Event Summary.

MESSAGE_AVAILABLE : Bit 4, value 16, of the Status Byte Register indicating Message Available.

REQUEST_SERVICE_SUMMARY : Bit 6, value 64, of the Status Byte Register indicating Service Request.

USER0 : Bit 0, value 1, of the Status Byte Register indicating User0.

USER1 : Bit 1, value 2, of the Status Byte Register indicating User1.

ERROR_EVENT_QUEUE_SUMMARY : Bit 2, value 4, of the Status Byte Register indicating Error/Event Queue Summary.

QUESTIONABLE_SUMMARY : Bit 3, value 8, of the Status Byte Register indicating Questionable Summary.

OPERATION_SUMMARY : Bit 7, value 128, of the Status Byte Register indicating Operation Summary.

NONE : None, value 0, indicates no register bits are set.
