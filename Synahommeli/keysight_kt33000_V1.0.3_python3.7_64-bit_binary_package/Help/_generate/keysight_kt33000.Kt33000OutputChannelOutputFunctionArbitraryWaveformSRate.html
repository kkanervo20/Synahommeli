property sampling_rate

Access

ReadWrite

Type

float

Remarks
•Parameter can be from 1 µSa/s to 250 MSa/s (EDU33210 series and 33500 Series) or 1 GSa/s (33600 Series), default 40 kSa/s. s. Limited to 62.5 MSa/s (EDU33210 series and 33500 Series) or 250 MSa/s (33600 Series) if InterpolationEnabled is OFF.
•Typical Return : +1.000000000000000E+04
•The sample rate and frequency parameter are not coupled when playing an arbitrary waveform segment. The concept of frequency does not apply for arbitrary waveform sequences.
•Setting a sample rate when not in the ARB mode will not change the frequency. For example, if the current function is sine, setting sample rate has no effect until the function changes to ARB.
•The maximum sample rate depends on the filter applied to the arbitrary waveform. See InterpolationEnabled for details.
