class Kt33000OutputChannelOutputSync

Kt33000OutputChannelOutputSync interface.

property enabled

Disables (OFF) or enables (ON) the front panel Sync connector.

Access

ReadWrite

Type

bool

Remarks
•Parameter : 0 (OFF) or 1 (ON). Default 1 (ON).
•Disabling the Sync signal reduces output distortion at lower amplitudes.
•When Sync is disabled, the output level on the Sync connector is a logic "low."
•Inverting a waveform (OutputPolarity set to Inverted) does not invert the Sync signal.
property mode

Specifies normal Sync behavior (Normal), forces Sync to follow the carrier waveform (Carrier), or indicates marker position (Marker).

Access

ReadWrite

Type

SyncMode

Remarks
•Parameter : Normal, Carrier or Marker. Default Normal.
•Typical Return : Normal, Carrier, or Marker.
•The following table details the command's behavior:
<mode> Sync Behaviour Conditions
Normal Sync follows envelope of burst signal. When BurstEnabled is set to ON.
Sync follows envelope of sweep signal. When SweepEnabled is set to ON.
Sync follows modulating signal. When modulating and modulation source is Internal.
Sync follows Function signal. All other conditions.
Carrier Sync follows current MarkerPoint setting. When BurstEnabled or SweepEnabled is set to ON and Function is Arbitrary.
Sync follows Function signal while BurstEnabled is set to ON. When BurstEnabled is set to ON and Function is not Arbitrary.
Sync follows Function signal. All other conditions.
Marker Sync follows current MarkerPoint setting. When in CW mode and Function is Arbitrary
When modulating, modulation source is internal, and Function is Arbitrary or <modulation> Internal function is Arbitrary
When modulating, modulation source is external, and Function is Arbitrary
Sync follows current MarkerCycle setting. When BurstEnabled is ON.
Sync follows current MarkerFrequency setting. When SweepEnabled is ON
Sync follows modulating signal. When modulating, modulation source is not external, Function is not Arbitrary, and internal function is not Arbitrary
Sync follows Function signal. All other conditions.
property polarity

Sets the desired output polarity of the Sync output to trigger external equipment that may require falling or rising edge triggers.

Access

ReadWrite

Type

SyncPolarity

Remarks
•Parameter : Normal or Inverted. Default Normal.
•Typical Return : Normal or Inverted.
•Normal: voltage on Sync output connector is near zero, and rises when a Sync event occurs. Voltage stays high (approximately 3.3 V into high impedance) until Sync signal is de-asserted, when it falls back to near zero.
•Inverted: opposite of NORMal.
•The Sync signal may be derived from either channel in a two-channel instrument (OutputSyncSource), and from several operating modes of the Sync signal (OutputSyncMode).
property source

Sets the source for the Sync output connector.

Access

ReadWrite

Type

SyncSource

Remarks
•Parameter : Channel1 or Channel2. Default Channel1.
•Typical Return : Channel1 or Channel2.
