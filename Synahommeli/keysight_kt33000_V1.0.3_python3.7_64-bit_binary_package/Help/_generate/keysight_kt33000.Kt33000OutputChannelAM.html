Kt33000OutputChannelAM

Access interface members using: driver.output_channels[<name_or_index>].am
Where driver is a reference to the driver’s root interface returned by the constructor.
class Kt33000OutputChannelAM

Kt33000OutputChannelAM interface.

property depth

Sets internal modulation depth (“percent modulation”) in percent.

Access

ReadWrite

Type

float

Remarks
•Parameter : 0 to 120, default 100.
•Typical Return : +5.000000000000000E+01
•Even at greater than 100% depth, the instrument will not exceed ±5 V peak on the output (into a 50 Ω load). To achieve modulation depth greater than 100%, output carrier amplitude may be reduced.
•With AMSource External, carrier waveform is modulated with an external waveform. The modulation depth is controlled by the ±5 V signal level (or the optional ±1 V signal level on the 33600 Series) on the rear-panel Modulation In connector.
For example, if modulation depth is 100%, then when the modulating signal is at +5 V (or optionally +1 V on the 33600 Series), the output will be at the maximum amplitude. Similarly, a -5 V modulating signal (or optionally -1 V on the 33600 Series) produces output at minimum amplitude.
property dssc_enabled

Selects Amplitude Modulation mode − Double Sideband Suppressed Carrier (ON) or AM modulated carrier with sidebands (OFF).

Access

ReadWrite

Type

bool

Remarks
•Parameter : 0 (OFF) or 1 (ON). Default 0 (OFF).
•In DSSC AM, zero modulation results in zero output signal, and increasing modulation input signal raises the amplitude of the sidebands in proportion to the amplitude of the modulating signal.
•In "normal" AM, zero modulation results in a half-amplitude carrier wave signal being output. As modulation input signal rises, the carrier is amplitude modulated between 0 and 100% amplitude.
•In DSSC, the AMDepth setting applies, and scales the modulation signal from 0 to 120% modulation.
property enabled

Enables or disables modulation.

Access

ReadWrite

Type

bool

Remarks
•Parameter : 0 (OFF) or 1 (ON). Default 0 (OFF).
•To avoid multiple waveform changes, enable modulation after configuring the other modulation parameters.
•Only one modulation mode may be enabled at a time.
•The instrument will not enable modulation with sweep or burst enabled. When you enable modulation, the sweep or burst mode is turned off.

property internal_frequency

Sets frequency of modulating waveform. The waveform chosen as modulating source will operate at that frequency, within waveform frequency limits.

Access

ReadWrite

Type

float

Remarks
•Input parameter can be from 1 μHz to the maximum allowed for the internal function. Default 100 Hz
•Typical Return : +1.000000000000000E+04
•When you select an arbitrary waveform as the modulating source, the frequency changes to the frequency of the arbitrary waveform, which is based on the sample rate and the number of points in the arbitrary waveform.
•This command should be used only with the internal modulation source (AMSource is set to Internal).
•When using an arbitrary waveform for the modulating source, changing this parameter also changes the cached metadata representing the arbitrary waveform's sample rate. You can also change the modulating frequency of an arbitrary waveform with ArbitraryWaveformFrequency, ArbitraryWaveformPeriod, and ArbitraryWaveformSamplingRate. These commands and the modulation frequency command are directly coupled in order to keep the arbitrary waveform behaving exactly as it was last played. If you later turn modulation off and select that same arbitrary waveform as the current function, its sample rate (and corresponding frequency based upon the number of points) will be the same as it was when played as the modulation source.
•If the internal function is Triangle, UpRamp, or DnRamp, the maximum frequency is limited to 200 kHz on the EDU33210 Series and 33500 Series or 800 kHz on the 33600 Series. If the internal function is PRBS, the frequency refers to bit rate and is limited.
property internal_function

Selects shape of modulating waveform.

Access

ReadWrite

Type

Waveform

Remarks
•Parameter: Sinusoid, Square, Ramp, NRamp, Triangle, Noise, PRBS or Arbitrary. Default Sinusoid.
•Typical Return : Sinusoid, Square, RAMP, NRamp, Triangle, Noise, PRBS, or Arbitrary.
•This command should be used only with the internal modulation source (AMSource Internal).
•Pulse and DC cannot be carrier waveform for AM.

property source

Selects the source of the modulating signal.

Access

ReadWrite

Type

ModulationSource

Remarks
•Parameter: Internal, External, Channel1,Channel2 or Both. Default Internal.
•Typical Return : Internal, External, Channel1, Channel2 or BOTH.
•If you select External, the carrier waveform is modulated with an external waveform.
•The modulation depth is controlled by the ±5 V signal level (or the optional ±1 V signal level on the 33600 Series) on the rear-panel Modulation In connector. For example, if modulation depth is 100%, then when the modulating signal is at +5 V (or optionally +1 V on the 33600 Series), the output will be at the maximum amplitude. Similarly, a -5 V modulating signal (or optionally -1 V on the 33600 Series) produces output at minimum amplitude.
•A channel may not be its own modulation source.
