StatusOperationFlags

class StatusOperationFlags

StatusOperationFlags enumeration with values corresponding to the 16-bit StatusOperation registers.

Members:

NONE : None, value 0, indicates no register bits are set.

CALIBERATING : Bit 0, value 1, of the Status Operation Register indicating Calibrating.

SELF_TEST : Bit 1, value 2, indicating a self-test is running.

INITIATE_CH1 : Bit 3, value 8, indicates Channel1 is initiated and outputting the desired waveform.

INITIATE_CH2 : Bit 4, value 16, indicates Channel2 is initiated and outputting the desired waveform.

TRIGGER_WAIT_CH1 : Bit 5, value 32, indicates that instrument is waiting for a trigger when channel1 is specied.

TRIGGER_WAIT_CH2 : Bit 6, value 64, indicates that instrument is waiting for a trigger when channel2 is specied.

CONFIG_EVENT : Bit 8, value 256, this bit is always 0 in the condition register, as it reflects an event, not a condition.

INSTRUMENT_LOCK : Bit 10, value 1024. If a remote interface (USB or LAN) has a lock, this bit will be set.

GLOBAL_ERROR : Bit 13, value 8192, this is set if any remote interface has an error in its error queue, and cleared otherwise.
