class Kt33000OutputChannelMarker

Kt33000OutputChannelMarker interface.

property cycle

Sets the marker cycle number at which the front panel Sync signal goes low in a burst mode operation.

Access

ReadWrite

Type

float

Remarks
•Parameter can be a whole number from 2 to number of cycles in the burst plus one (NCYCles+1), default 2.
•Typical Return : +2.000000000000000E+03.
•This is valid only if burst is enabled and OutputSyncMode is Marker.
•With burst enabled, the marker cycle must be less than or equal to the number of cycles in the burst plus one. Attempting to set the marker cycle outside this range will set marker cycle equal to middle of burst. From the remote interface, a "Settings conflict" error will also be generated.
property enabled

Marker state enabled (ON) or disabled (OFF) for the specified channel. When the frequency marker is disabled, the signal output from the Sync connector is the normal Sync signal for the carrier waveform.

Access

ReadWrite

Type

bool

Remarks
•Parameter : 0 (OFF) or 1 (ON). Default 0(OFF).
•This command is supported on 33210 series only.
•The OutputSyncEnabled command is overridden by the setting of the MarkerEnabled command. Therefore, when the marker frequency is enabled (and the sweep mode is also enabled), the OutputSyncEnabled command is ignored.
property frequency

Sets the marker frequency in Hertz(Hz) at which the front panel Sync signal goes low during a sweep.

Access

ReadWrite

Type

float

Remarks
•Parameter can be any frequency between start and stop frequency, default 500 Hz
•Typical Return : +2.000000000000000E+03
•This is valid only if sweep is enabled and OutputSyncMode is Marker.
•When sweep is enabled, marker frequency must be between start frequency and stop frequency. Attempting to set the marker cycle outside this range will set marker frequency to start frequency or frequency (whichever is closer). From the remote interface, a "Settings conflict" error will also be generated.

property point

Sets the sample number at which the front panel Sync signal goes low within the active arbitrary waveform.

Access

ReadWrite

Type

int

Remarks
•Parameter can be a whole number from 4 to number of samples in waveform, minus 3; default is midpoint of arbitrary waveform.
•Typical Return : +1.000000000000000E+01
•Command only sets marker point in currently active arbitrary waveform (SelectArbitraryWaveform), not in a sequence.
•Command is valid only under these conditions:
•OutputSyncMode set to Marker, Function set to Arbitrary, FrequencyMode set to CW.
•OutputSyncMode set to Carrier, Function set to Arbitrary, BurstEnabled is ON.
•OutputSyncMode set to Carrier, Function set to Arbitrary, FrequencyMode set to Sweep.
•OutputSyncMode set to Marker, internal modulation active, and either Function set to Arbitrary or a modulating waveform's internal function is set to Arbitrary.
•OutputSyncMode set to Marker, external modulation active, and Function set to Arbitrary.
