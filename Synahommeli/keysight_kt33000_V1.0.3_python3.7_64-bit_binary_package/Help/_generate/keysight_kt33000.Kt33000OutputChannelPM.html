Kt33000OutputChannelPM

Access interface members using: driver.output_channels[<name_or_index>].pm
Where driver is a reference to the driver’s root interface returned by the constructor.
class Kt33000OutputChannelPM

Kt33000OutputChannelPM interface.

property deviation

Sets the phase deviation in degrees. This value represents the peak variation in phase of the modulated waveform from the carrier waveform.

Access

ReadWrite

Type

float

Remarks
•Parameter : 0 to 360, default 180.
•Typical Return : +9.000000000000000E+01.
•With the External modulating source, deviation is controlled by the ±5 V signal level (optionally ±1 V on the 33600 Series) on the rear-panel Modulation In connector. For example, if you have set the frequency deviation to 180 degrees, then a +5 V signal level (optionally ±1 V on the 33600 Series) corresponds to a +180 degree phase deviation. Lower external signal levels produce less deviation, and negative signal levels produce negative deviation.
property enabled

Sets the PM state (Enabled or Disabled) for the specified channel.

Access

ReadWrite

Type

bool

Remarks
•Parameter : 0 (OFF) or 1 (ON). Default 0 (OFF).
•To avoid multiple waveform changes, enable modulation after configuring the other modulation parameters.
•Only one modulation mode may be enabled at a time.
•The instrument will not enable modulation with sweep or burst enabled. When you enable modulation, the sweep or burst mode is turned off.

property internal_frequency

Sets the frequency of the modulating waveform.

Access

ReadWrite

Type

float

Remarks
•Parameter : 1 μHz to the maximum allowed for the internal function. Default 10 Hz.
•Typical Return : +1.000000000000000E-06
•When you select an arbitrary waveform as the modulating source, the frequency changes to the frequency of the arbitrary waveform, which is based on the sample rate and the number of points in the arbitrary waveform.
•This command should be used only with the internal modulation source (i.e PMSource is set to Internal).
•When using an arbitrary waveform for the modulating source, changing this parameter also changes the cached metadata representing the arbitrary waveform's sample rate. You can also change the modulating frequency of an arbitrary waveform with ArbitraryWaveformFrequency, ArbitraryWaveformPeriod, and ArbitraryWaveformSamplingRate. These commands and the modulation frequency command are directly coupled in order to keep the arbitrary waveform behaving exactly as it was last played. If you later turn modulation off and select that same arbitrary waveform as the current function, its sample rate (and corresponding frequency based upon the number of points) will be the same as it was when played as the modulation source.
•If the internal function is Triangle, UpRamp, or DnRamp, the maximum frequency is limited to 200 kHz on the EDU33210 Series and 33500 Series or 800 kHz on the 33600 Series. If the internal function is PRBS, the frequency refers to bit rate and is limited.
property internal_function

Selects shape of modulating waveform.

Access

ReadWrite

Type

Waveform

Remarks
•Parameter: Sinusoid, Square, Ramp, NRamp, Triangle, Noise, PRBS or Arbitrary. Default Sinusoid.
•Typical Return : Sinusoid, Square, Ramp, NRamp, Triangle, Noise, PRBS, or Arbitrary.
•This command should be used only with the internal modulation source(i.e PMSource is set to Internal).
•You can use noise as the modulating waveform, but you cannot use noise, pulse, or DC as the carrier.

property source

Selects the source of the modulating signal for the specified channel.

Access

ReadWrite

Type

ModulationSource

Remarks
•Parameter: Internal, External, Channel1 or Channel2. Default Internal.
•Typical Return : Internal, External, Channel1, or Channel2.
•If you select External, the carrier waveform is modulated with an external waveform.
•With the External modulating source, deviation is controlled by the ±5 V signal level (optionally ±1 V on the 33600 Series) on the rear-panel Modulation In connector. For example, if you have set the frequency deviation to 180 degrees, then a +5 V signal level (optionally ±1 V on the 33600 Series) corresponds to a +180 degree phase deviation. Lower external signal levels produce less deviation, and negative signal levels produce negative deviation.
•A channel may not be its own modulation source.
