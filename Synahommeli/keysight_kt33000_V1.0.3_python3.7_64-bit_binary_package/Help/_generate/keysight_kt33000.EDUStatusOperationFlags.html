EDUStatusOperationFlags

class EDUStatusOperationFlags

EDUStatusOperationFlags enumeration with values corresponding to the 16-bit StatusOperation registers.

Members:

CALIBRATING : Bit 0, value 1, of the Status Operation Register indicating Calibrating.

NONE : None, value 0, indicates no register bits are set.

SELF_TEST : Bit 1, value 2, indicating a self-test is running.

INITIATED_CHANNEL1 : Bit 3, value 8, indicates Channel1 is initiated and outputting the desired waveform.

INITIATED_CHANNEL2 : Bit 4, value 16, indicates Channel2 is initiated and outputting the desired waveform.

WAIT_FOR_TRIGGER_CHANNEL1 : Bit 5, value 32, indicates that instrument is waiting for a trigger when channel1 is specied.

WAIT_FOR_TRIGGER_CHANNEL2 : Bit 6, value 64, indicates that instrument is waiting for a trigger when channe2 is specified.

CONFIGURATION_CHANGED : Bit 8, value 256, this bit is always 0 in the condition register, as it reflects an event, not a condition.

GLOBAL_ERROR : Bit 13, value 8192, this is set if any remote interface has an error in its error queue, and cleared otherwise.
