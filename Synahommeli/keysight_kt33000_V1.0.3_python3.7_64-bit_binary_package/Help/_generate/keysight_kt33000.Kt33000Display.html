Kt33000Display

Access interface members using: driver.display
Where driver is a reference to the driver’s root interface returned by the constructor.
class Kt33000Display

Kt33000Display interface.

property arb_rate_unit

Specifies whether the rate units for arbitrary waveforms are samples per second, (SRate), Hz (Frequency) or seconds (Period). Default SRate.

Access

ReadWrite

Type

DisplayARBRateUnits

Remarks
This command is supported on 33500 series and 33600 series Function Generator only.
clear_text(self: keysight_kt33000.keysight_kt33000.Kt33000Display) → None

Clears the text message from the front panel display.

Returns

None

Remarks
•This command is not supported on EDU33210 series.
•With DisplayEnabled ON, this command returns the display to its normal mode.
•With DisplayEnabled OFF, this command clears the message and the display remains disabled.
•The display text is unaffected by Reset command. It is cleared at power-on.
property enabled

Disables or enables the front panel display.

Access

ReadWrite

Type

bool

Remarks
•Parameter : 0 (OFF) or 1 (ON). Default 1 (ON).
•When disabled, the front panel display turns black, and all annunciators are disabled. However, the screen remains on.
•Disabling the display improves command execution speed from the remote interface and provides basic security.
•The display is enabled when power is cycled, or when you return to local (front panel) operation by pressing the front panel [Local] key.
property focus

Selects the channel that is displayed “in front” on a two-channel instrument. You can select either Channel1 or Channel2. Default Channel1.

Access

ReadWrite

Type

DisplayFocus

Remarks
This command is supported on 33500 series and 33600 series Function Generator only.
get_screen_dump(self: keysight_kt33000.keysight_kt33000.Kt33000Display, data_format: keysight_kt33000.keysight_kt33000.HCopyDataFormat) → numpy.ndarray[uint8]

Returns the front panel display image (“screen shot”) of specified image format.

Parameters

data_format (HCopyDataFormat) – Data format can be either PNG or BMP.

Returns

Typical Return : A definite-length binary block containing the image.

Return type

numpy.ndarray<with dtype:uint8>

Remarks
Definite-length Block Data: Definite-length block data allows any type of device-dependent data to be transmitted as a series of 8-bit binary data bytes. This is particularly useful for transferring large quantities of data or 8-bit extended ASCII codes.

property sweep_range_mode

Selects the method for specifying sweep frequency range. You can specify the range as Start and Stop values (StartStop) or as a Span around a Center value (CenterSpan). Default StartStop.

Access

ReadWrite

Type

DisplaySweepRange

Remarks
•This command is supported on 33500 series and 33600 series Function Generator only.
•To sweep up in frequency, set a positive frequency span; to sweep down, set a negative frequency span.
property text

Displays a text message on the front panel display.

Access

ReadWrite

Type

str

Remarks
•Parameter : Quoted string of up to 40 standard keyboard characters, default "".
•This command is not supported on EDU33210 series.
•Sending this command overrides the display state. You can display a message with the display disabled.
•While a message is displayed, information relating to the current instrument operation is not sent to the front panel display.
•The display text is unaffected by Reset command. It is cleared at power-on.
property view

Selects the screen layout. It can be Standard View (Standard), Large Text (Text), Large Graph (Graph) or Channel1/Channel2 (Dual). Default Standard.

Access

ReadWrite

Type

DisplayView

Remarks
This command is supported on 33500 series and 33600 series Function Generator only.
