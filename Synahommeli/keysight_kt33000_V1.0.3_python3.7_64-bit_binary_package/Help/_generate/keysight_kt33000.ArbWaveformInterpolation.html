ArbWaveformInterpolation

class ArbWaveformInterpolation

Normal, Step or Off.

Members:

NORMAL : Normal filters the data points with the filter that provides the flattest frequency response. This effectively smoothes the signal, but sharp transitions will have preshoot and overshoot.

STEP : Step filters the data points in a way that effectively smoothes the signal while minimizing the preshoot and overshoot. This is default Arbitrary waveform Interpolation.

OFF : Off steps from point to point at the sample rate. Moves between data points are accomplished as quickly as possible with no smoothing.
