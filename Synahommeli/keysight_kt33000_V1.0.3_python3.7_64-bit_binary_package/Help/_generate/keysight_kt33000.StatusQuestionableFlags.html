StatusQuestionableFlags

class StatusQuestionableFlags

StatusQuestionableFlags enumeration with values corresponding to the 16-bit StatusQuestionable registers.

Members:

VOLT_CH1 : Bit 0, value 1, indicates voltage overload on channel 1 output connector. The output has been disabled.

NONE : None, value 0, indicates no register bits are set.

VOLT_CH2 : Bit 1, value 2, indicates voltage overload on channel 1 output connector. The output has been disabled.

UNLOCKED_LOOP : Bit 5, value 32, indicates that function generator has lost phase lock. Frequency accuracy will be affected.

CALIBRATION_ERROR : Bit 8, value 256, of the Status Questionable Register indicating error occurred during calibration, calibration is unsecured, or calibration memory has been lost.

EXTERNAL_REF : Bit 9, value 512, indicating External timebase has been detected.
