PulseHold

class PulseHold

Hold either pulse width or duty cycle.

Members:

WIDTH : On selecting Width, the instrument holds the pulse width setting (in seconds) constant as the period is varied. This is default Pulse Hold parameter.

DUTY_CYCLE : On selecting DutyCycle, the instrument holds the pulse duty cycle setting (in percent) constant as the period is varied.
